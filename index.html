1991-3, World Wide Web 

The first Web page was created in 1991. Then in 1993, Mosaic a web browser released. 
This was important because it was the first browser that made the Internet easy for ordinary people to access. After the Mosaic was released, 
a year later Netscape Navigator was created, which competed with Mosaic. Also Netscape made online transactions secure which fueled the start of 
companies like Amazon.com,and eBay. After a few years later in 1998, Google was created, it changed how people found information online,
and we still use it to this day.
